# java-movie-precourse

객체 지향 설계 원칙을 기반으로, 순수 Java로 도메인을 구현한 뒤
데이터베이터베이스 연동과 Spring Boot를 통한 HTTP API까지 확장하면서도
객체 지향적 사고가 흔들리지 않음을 직접 경험해보기

## 코딩 컨벤션
1. 들여쓰기는 4칸, 깊이는 최대 2
2. 메서드의 행은 15줄까지
3. else, switch 금지, use return
4. 3항 연산자 금지
